Responsive Accordion is the most easiest drag & drop accordion builder for WordPress. You can add multiple accordion and collapse with this.

== Description ==

Responsive Accordion is highly customizable accordion builder for wordpress. You can add unlimited accordion and collapse with unlimited colour. Responsive Accordion also support Font Awesome Icon.
You can easy sort your accordion position using backend. Responsive Accordion is built-in bootstrap. 

= Responsive Accordion Shortcode =

> `[WPSM_AC id=xx]`

Here xx is your accordion shortcode id.

== Installation ==

1. Upload the entire `responsive-accordion-and-collapse` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Responsive Accordion menu to customize your Responsive Accordion plugin.
4. Copy your publish accordion shortcode and then paste into your page or post to display your accordion
